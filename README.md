# 🏥 Prescripto – Doctor Appointment Booking System

A full-stack MERN application that simplifies the process of booking doctor appointments online. The platform enables users to browse doctors by specialty, view doctor profiles, select available time slots, and book appointments through an intuitive and responsive interface.

## 📌 Features

### 👨‍⚕️ Patient Features
- Browse doctors by specialty
- View detailed doctor profiles
- Book appointments by selecting date and time slots
- Responsive user interface
- Authentication and authorization
- Manage booked appointments

### 🩺 Doctor Features
- Doctor login
- View upcoming appointments
- Manage appointment status
- Dashboard for daily activities

### 👨‍💼 Admin Features
- Secure admin login
- Add new doctors
- Manage doctor information
- View platform statistics
- Manage appointments

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/SRIKANTH-0001/Prescripto.git
```

### Navigate into the project

```bash
cd Prescripto
```

### Install frontend dependencies

```bash
cd frontend
npm install
```

### Install backend dependencies

```bash
cd ../backend
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm run dev
```
